# Primary Navigation

Orbit Axis now uses six primary sections with the chatbot as the center action:

- Home
- Me
- Tarot
- Ask Orbit Axis
- Learn
- News
- More

Mobile order is:

Home | Me | Tarot | Ask | Learn | News | More

The Ask control is larger, raised above the bottom navigation, and uses an orbit-style icon instead of a generic robot. History is no longer a primary navigation item and is available through More.

Desktop keeps the same hierarchy in the left rail:

Home
Me
Tarot
Ask Orbit Axis
Learn
News
More

Primary navigation labels remain visible where space allows. Mobile keeps text labels for all seven tap targets.
