# Orbit Axis Interface Direction

The interface direction is calm, soft, space-themed, modern, beginner-friendly, and readable for nontechnical users.

Design choices:

- deep-space base colors with moon white, pale lavender, restrained blue, and subtle blush accents
- translucent dark surfaces with soft borders
- central Ask Orbit Axis action
- single-column mobile pages
- plain-language labels before technical astrology
- progressive disclosure for exact placements and advanced details

The redesign preserves the deterministic fortune engine, current-sky calculations, Moon phase data, My Chart calculations, Simple / Balanced / Advanced detail modes, Fortune History, Local Intelligence, and existing API routes.

Unfinished product areas use honest empty states rather than fabricated content.
