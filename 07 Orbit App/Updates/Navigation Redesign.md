# Navigation Redesign

Implemented the Orbit Axis primary navigation redesign around a central chatbot.

Changed:

- primary navigation is now Home, Me, Tarot, Ask, Learn, News, More
- Ask Orbit Axis is visually centered and elevated on mobile
- desktop rail follows the same hierarchy
- Home remains the daily return point
- Me contains personal chart calculations and progressive detail
- History moved under More
- Tarot, Learn, and News have honest shell states
- technical Local Intelligence diagnostics moved into More -> Settings -> Local Intelligence

Not implemented in this update:

- full tarot reading system
- Learn course content and progress tracking
- live News aggregation
- full saved-chart manager UI
- full synastry interface
