# Code of conduct

## Our standard

Be respectful and constructive. Assume good faith. Critique code and ideas,
not people.

Unacceptable behaviour includes harassment, personal attacks, discriminatory
language, and publishing others' private information.

Astrology attracts strong opinions in both directions. Debate about whether
astrology is meaningful is welcome in the abstract, but this project's technical
spaces are for building the software. Dismissing contributors personally for
their interest in the subject is not acceptable; nor is asserting that the
software predicts real events, which it does not.

## Scope

Applies to issues, pull requests, discussions, and any space where someone
represents the project.

## Enforcement

Report concerns privately via GitHub to the repository owner (`EZmannBuilds`).
Reports are handled confidentially and on a best-effort basis. Maintainers may
edit, hide, or remove contributions that violate this document, and may block
accounts for repeated or severe violations.

This is a small personal project, not an organisation with a moderation team.
Expect a human response rather than a formal process.
